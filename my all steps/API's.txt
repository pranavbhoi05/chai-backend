###############################################################

| Term       | Meaning                                    |
| ---------- | ------------------------------------------ |
| **API**    | Way to access features/data of your app    |
| **Route**  | A specific API (like `/videos/:id`)        |
| **Client** | Who calls the API (browser, Postman, app)  |
| **Server** | Who responds to the API (your Express app) |
