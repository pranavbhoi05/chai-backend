

Node.js is the engine that lets you run JavaScript outside the browser — on your computer or server.



✅ What is Mongoose?
* Mongoose is a Node.js library that helps you work with MongoDB easily.

It lets you:
* Define schemas (structure) for your data
* Run MongoDB queries using easy JavaScript code
* Add validation, hooks, and relationships between data

🔧 Why use Mongoose?
*MongoDB by default is schemaless (no fixed structure). That’s powerful, but can lead to mistakes.
*Mongoose gives you structure and control.

🧠 First: What does “JavaScript on servers” mean?

#Usually, JavaScript runs in the browser (frontend) — like:
*Buttons clicking
*Animations
*Form validation


# But with Node.js, we can run JavaScript on the server — which means:
*Storing and retrieving data from a database
*Handling login/signup
*Sending responses to users (HTML, JSON, etc.)
*Controlling what a website or app does behind the scenes
